API de Dados

Essa página descreve a API de Dados, uma forma automatizada de obter acesso aos modelos preditivos de retenção e evasão de alunos, de forma geral seu uso possibilita integrar essas informações a outros sistemas. Por esse motivo, os métodos têm CORS configurado de maneira a permitir chamadas de outros domínios. A documentação abaixo descreve, com exemplos, como utilizar a interface de dados.

Acesso API de evasão

O acesso a API de dados é público, não é necessario criar uma conta tampuco autenticar.

Estrutura das Requisições

URL

É necessário fazer uma chamada HTTP, método GET, para URL http:127.0.0.1:5000/api/ predict / method

Parâmetros de URL

Dois parâmetros são obrigatórios e presentes em todas requisições:

predict: Qual predição se quer fazer
Tipo: String
Domínio de dados
evasion : para previsão de evasão
retention : para previsão de retenção

method: Metódo da API de Dados requisitado
Tipo: String
Domínio de dados
student : consulta por estudante
period : consulta por ano.periodo
course : consulta por curso
