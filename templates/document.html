ePredict API

API de Dados

Essa página descreve a API de Dados, uma forma automatizada de obter acesso aos modelos preditivos de evasão de alunos de graduação, de forma geral seu uso possibilita integrar essas informações a outros sistemas. Por esse motivo, os métodos têm CORS configurado de maneira a permitir chamadas de outros domínios. A documentação abaixo descreve, com exemplos, como utilizar a interface de dados.

Acesso API de evasão

O acesso a API de dados é público, não é necessario criar uma conta tampuco autenticar.

Estrutura das Requisições

URL

É necessário fazer uma chamada HTTP, método GET, para URL http://epredictapi.herokuapp.com/ predict / method

Parâmetros de URL

Exsitem dois parâmetros presentes nas requisições:

predict: Qual predição se quer fazer
Tipo: String
Domínio de dados
evasao : para predição de evasão
retencao : para predição de retenção (não implementado)

method: Metódo da API de Dados requisitado
Tipo: String
Domínio de dados
curso : consulta por curso
aluno : consulta por aluno

Repostas

As resposta são retornadas no formato JSON.

Exemplo de URL

            http://epredictapi.herokuapp.com/evasao/curso/es

Métodos da API

curso

Descrição

Retorna informações quantitativas e predições de evasão nos cursos.

Resultado
percent_evasao_cc: Percentual de evasão no curso de Ciências da Computação
Tipo: Decimal
percent_evasao_es: Percentual de evasão no curso de Engenharia de Software
Tipo: Decimal
percent_evasao_si: Percentual de evasão no curso de Sistemas da Informação
Tipo: Decimal
percent_evasao_total: Percentual de evasão em todos os cursos
Tipo: Decimal
quant_aluno_cc: Quantidade de alunos no curso de Ciências da Computação
Tipo: Decimal
quant_aluno_es: Quantidade de alunos no curso de Engenharia de Software
Tipo: Decimal
quant_aluno_si: Quantidade de alunos no curso de Sistemas de Informação
Tipo: Decimal
quant_aluno_total: Quantidade total de alunos
Tipo: Decimal
quant_evasao_cc: Quantidade de predições de evasão no curso de Ciências da Computação
Tipo: Decimal
quant_evasao_es: Quantidade de predições de evasao no curso de Engenharia de Software
Tipo: Decimal
quant_evasao_si: Quantidade de predições de evasão no curso de Sistemas de Informação
Tipo: Decimal
quant_evasao_total: Quantidade total de predições de evasão
Tipo: Decimal

Exemplo de chamada

            http://epredictapi.herokuapp.com/evasao/
